## Development server

> Run front-end only

```
npm run ng-dev
```

> Run server only

```
npm run server
```

> Run project

```
npm run build
```

Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

## Build

```
npm start
```
